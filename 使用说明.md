# 计算机网络复习宝典 APK 使用说明

## APK 位置

`E:\Learning\课程学习\计算机网络\计算机网络_考试练习_APK\build\out\计算机网络复习宝典.apk`

## 当前版本

- 版本：`2.10.0`
- 题量：372 题
- 题型：判断题、单选题、多选题、填空题
- 章节：第 1 章到第 7 章

## 本次解析优化

- 章节知识卡片升级为独立 `chapter_cards.json` 数据，24 张知识点卡覆盖 372/372 题。
- 每张卡片新增“底层知识点、必须原样背 / 会算、易错开关、逐题覆盖线索”，避免只抽前几条解析导致漏点。
- 子代理分章审查后补入关键标准词和数字，例如 `129.7.0.0`、`1011011011000011`、`802.11b`、`分组过滤路由器`、`TCP 80`、`4 RTT`、`0.0.0.0 / 255.255.255.255` 等。
- 372 题全部重写为“本题优先”的解析结构。
- 多选题不再只逐项判对错，新增“先定规则、应选组、排除组、为什么是多选”。
- 单选题强化正确项和干扰项边界。
- 判断题强化“错在哪里、怎么改才对”。
- 填空/计算/流程题强化“题眼、公式/规则、步骤、标准填法、易错词形”。
- 知识点详解改成“本题考什么、底层模型、本题步骤、选项/空格/判断句拆解、相邻概念边界、小白复述”。
- 已修正已发现的错配题，例如 Web/ARPANET、PPP/P2P、数据链路层功能、Web 访问协议、TCP 确认号、UDP、SMTP、防火墙等。

## 功能概览

- 离线刷 372 题。
- 支持按题型和章节筛选。
- 支持全部题、错题本、随机刷题、章节知识卡片。
- 左右滑动切换上一题/下一题，答案页也支持横滑。
- 答错自动加入错题本；普通刷题答对不会自动移出错题本。
- 错题本模式中连续答对达到设置次数后才移出错题本，次数可在设置中调整。
- 可导出当前题，自动复制并打开系统分享面板，方便发给 QQ、微信或 AI 工具继续追问。
- 错题本中可导出错题 Markdown。
- 章节知识卡片可在 App 内直接打开，正反面卡片式复习。
- 支持通过 GitHub Release 联网检查更新、下载新 APK、调起系统安装，并在安装完成后清理缓存安装包。

## 安装方式

1. 把 `计算机网络复习宝典.apk` 发送到手机。
2. 在手机文件管理器、QQ、微信或其他应用中打开 APK。
3. 如果系统提示未知来源安装，允许当前应用安装一次即可。
4. 安装完成后，桌面应用名为“计算机网络复习宝典”。

## 构建校验口径

- APK 包名：`com.dz.networkquiz`
- 最低 Android：Android 6.0 / API 23
- 目标 Android：Android 15 / API 35
- 题库校验：372 题、5 张图片题、7 章、24 个知识点分组

## 联网更新配置

1. 打开 App 底部的“设置”。
2. 默认更新源已经内置为 `ZhiKong0/review-baodian-apk`，一般不需要手动配置。
3. App 打开且联网后会自动检查更新；也可以手动点击“检查更新”。
4. 如果你以后迁移到别的 GitHub 仓库，再点击更新源配置并输入新的 `owner/repo` 或 GitHub 仓库链接。
5. 如果发现新版本，点击下载安装，系统安装完成后 App 会自动清理缓存安装包。
